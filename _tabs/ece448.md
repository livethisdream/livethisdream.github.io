<p class="course-tagline" style="color:#666;margin-top:-0.5em;">Introduction to Software Defined Radios &middot; Spring 2026</p>

This course is a **hands-on introduction to Software Defined Radios (SDRs)**.
We use **GNURadio** to build real flowgraphs against real signals — analog and
digital modulation, link budgets, custom DSP blocks, RF reverse engineering,
and live-signal decoding (ADS-B, AIS, HD Radio). It culminates in a
**self-defined final project** where you set your own success criteria and
defend them.
